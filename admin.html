Complete Guide — How to Add a Team Member
1
Click Add Member above → fill in their name, role, password and permissions → click Generate Config.
2
Click ⬇ Download Config on the member card — it downloads a .txt file with the exact code block to paste.
3
Open school_data.js in your code editor (VS Code, Notepad++, etc.).
4
Find the STAFF_MEMBERS = [ array — it starts around line 43. Paste your downloaded code block inside the [ ] brackets, before the closing ];
5
Save the file → run git add . → git commit -m "add member" → git push → GitHub Pages updates in ~60 seconds → member can log in immediately.
✓
Data sync: Owner clicks Export → downloads JSON → sends via WhatsApp/Drive → member opens admin panel → clicks Import → all data appears instantly.
